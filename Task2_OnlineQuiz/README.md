# Reflex Rumble - Online Quiz Application

Reflex Rumble is an interactive quiz application built using **Python** and **Tkinter**. The project allows users to sign up, log in, choose a quiz category, answer multiple-choice questions, and track their scores. The application stores user information and quiz results, creating a personalized quiz experience.

## ✨ Features

* User Registration and Login System
* Secure user data storage using JSON
* Category-based quiz selection
* Multiple-choice questions
* Automatic score calculation
* Score display at the end of each quiz
* User progress and score tracking
* Simple and user-friendly graphical interface
* Customized and updated version designed specifically for internship requirements

## 🛠️ Technologies Used

* Python 3
* Tkinter (Python's standard GUI library)
* JSON for data storage

## 🚀 How to Run

1. Make sure Python is installed on your system.
2. Clone this repository or download the source code.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python ReflexRumble.py
```
